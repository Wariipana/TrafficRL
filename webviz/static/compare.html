Cómo leerlo: menor espera (s) y mayor throughput = mejor. La columna “Mejora vs fijo” compara la espera media contra el semáforo de tiempo fijo: verde = el RL reduce la espera, rojo = es peor. El ★ marca el mejor por menor espera. Genera estos datos con bash scripts/train_all.sh o python -m rl.training.benchmark.
